OBJECT DETECTION:
When passed an image, the program can identify where the animals are located in this image and draw boxes around them
IMAGE CLASSIFICATION:
When passed those cropped photos of the animals from the original image, the model can classify the type of animal that it is
RESULT:
The program will return the original image with red boxes drawn around the animals and labels printed within each box.
